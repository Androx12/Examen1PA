Este programa se ejecuta desde el modulo main.py y cuenta con un solo directorio
El modulo main realiza la distribución completa de los demas modulos incluyendo las VISTAS Y EL PATRON OBSERVATIONAL

****** El programa realiza lo siguiente:

Supongamos que es una fiesta y dentro de esa fiesta se va a regalar un pemio totalmente aleatorio entre todos los participantes que ingresen.

Cada participante al ingresar debera mencionar su nombre al encargado para que este lo pueda registrar, además en caso de error el encargado tendrá la opción de eliminar nombres de la lista. Esto para evitar que existan nombres duplicados o bien por alguna falta ortografica o UN NOMBRE VACIO. Ademas al final realizara un sorteo aleatorio eligiendo a un solo participante registrado dentro de la fiesta.

FUNCIONALIDADES

1 REGISTRAR (Crea LinkedList).

2 Eliminar (Elimina dato ademas vuelve a crear la estructura de nodos y punteros para reestablecer la continuidad de la lista).

3 Realizar un sorteo aleatorio contando como maximo de randon la cantidad de participantes registrados.

4 El patron Observational está implementado para realizar una notificación de cambios a los observadores (Que se incluyen de forma manual una vez finalizado el programa del premio) Este patron esta realizado mediante parametros de tiempo importando la libreria TIME y no por cantidades a manera de ejemplificar.

****** El programa realiza validaciones como:

Obligar al usuario a navegar UNICAMENTE por las opciones del MENU
En el momento que solicita un numero de INDEX para consulta o para eliminar un participante se obliga a ingresar numeros enteros

OTRAS NOTAS:
Solo ejecutar desde el MAIN
